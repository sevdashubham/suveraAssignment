# news-service
A serverless app of common endpoints

Run `yarn` or `npm install` to get started. You'll need the [serverless cli tool](https://serverless.com/framework/docs/providers/aws/cli-reference/) to develop offline with `serverless-offline`

Running locally:

```sls offline start --stage dev```

## Testing with Jest

```npm install -g jest```

## Run Tests

```jest --coverage```
